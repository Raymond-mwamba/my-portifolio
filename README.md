## ğŸš€ Getting Started

### Prerequisites
- Node.js 18 or higher
- npm or yarn

### Installation
1. Clone the repository
```bash
git clone https://github.com/Raymond-Mwamba/my-portfolio.git
cd my-portfolio
```

2. Install dependencies
```bash
npm install
```

3. Start development server
```bash
npm run dev
```

4. Build for production
```bash
npm run build
```
